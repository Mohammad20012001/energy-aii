# 🚀 دليل تكامل Groq API مع Energy.AI

## 📋 نظرة عامة

تم إضافة دعم كامل لـ Groq API إلى موقع Energy.AI، مما يوفر للمستخدمين خيار استخدام نماذج الذكاء الاصطناعي المتقدمة من Groq بدلاً من أو بالإضافة إلى Google Gemini.

## ✨ الميزات الجديدة

### 🤖 نماذج Groq المدعومة:
- **Llama 3.1 405B** - الأقوى والأكثر دقة
- **Llama 3.1 70B** - متوازن وموصى به
- **Llama 3.1 8B** - سريع ومناسب للاستخدام العادي
- **Mixtral 8x7B** - متعدد اللغات ومتخصص
- **Gemma 7B & 2 9B** - نماذج Google المحسنة

### ⚙️ إعدادات متقدمة:
- **Temperature** - التحكم في الإبداع (0.0 - 2.0)
- **Max Tokens** - الحد الأقصى للكلمات (256 - 4096)
- **Top P** - دقة التركيز (0.1 - 1.0)
- **تاريخ المحادثة** - حفظ واستعادة المحادثات
- **تقدير التكلفة** - حساب تقريبي للتكلفة

## 🔧 كيفية الإعداد

### 1. الحصول على مفتاح Groq API:

1. انتقل إلى [console.groq.com](https://console.groq.com)
2. أنشئ حساب جديد أو سجل الدخول
3. انتقل إلى قسم "API Keys"
4. أنشئ مفتاح API جديد
5. انسخ المفتاح (يبدأ بـ `gsk_`)

### 2. إعداد المفتاح في الموقع:

1. افتح موقع Energy.AI
2. افتح نافذة الدردشة
3. اختر "Groq AI" من القائمة المنسدلة
4. انقر على أيقونة الإعدادات ⚙️
5. أدخل مفتاح API في الحقل المخصص
6. انقر "حفظ الإعدادات"

## 🎯 كيفية الاستخدام

### التبديل بين مقدمي الخدمة:
```
1. افتح نافذة الدردشة
2. في شريط العلوي، ستجد قائمة منسدلة
3. اختر بين "Google Gemini" أو "Groq AI"
4. سيتم التبديل فوراً
```

### إعداد النموذج المفضل:
```
1. انقر على أيقونة الإعدادات ⚙️
2. اختر النموذج من قائمة "اختيار النموذج"
3. اضبط الإعدادات المتقدمة حسب الحاجة
4. احفظ الإعدادات
```

### اختبار الاتصال:
```
1. في نافذة الإعدادات
2. انقر "اختبار الاتصال"
3. انتظر النتيجة
4. إذا نجح، ستظهر رسالة تأكيد ✅
```

## 🔧 الإعدادات المتقدمة

### Temperature (الإبداع):
- **0.0 - 0.3**: ردود محافظة ودقيقة
- **0.4 - 0.7**: متوازن (موصى به)
- **0.8 - 1.2**: إبداعي ومتنوع
- **1.3 - 2.0**: إبداعي جداً (قد يكون غير دقيق)

### Max Tokens (الحد الأقصى للكلمات):
- **256**: ردود قصيرة
- **512**: ردود متوسطة
- **1024**: ردود مفصلة
- **2048**: ردود شاملة (موصى به)
- **4096**: ردود طويلة جداً

### Top P (دقة التركيز):
- **0.1 - 0.3**: تركيز عالي
- **0.4 - 0.7**: متوازن
- **0.8 - 0.9**: متنوع (موصى به)
- **1.0**: أقصى تنوع

## 💡 نصائح للاستخدام الأمثل

### 1. اختيار النموذج المناسب:
- **للحسابات الدقيقة**: Llama 3.1 405B
- **للاستخدام اليومي**: Llama 3.1 70B
- **للردود السريعة**: Llama 3.1 8B
- **للغات متعددة**: Mixtral 8x7B

### 2. ضبط الإعدادات:
- **للحسابات**: Temperature = 0.1-0.3
- **للشرح**: Temperature = 0.5-0.7
- **للإبداع**: Temperature = 0.8-1.0

### 3. توفير التكلفة:
- استخدم النماذج الأصغر للمهام البسيطة
- قلل Max Tokens عند عدم الحاجة لردود طويلة
- استخدم التخزين المؤقت للأسئلة المتكررة
